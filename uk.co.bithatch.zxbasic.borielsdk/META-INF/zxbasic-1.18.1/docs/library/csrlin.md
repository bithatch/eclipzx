# CSRLIN

## Syntax

```
CSRLIN()
```

## Description

Returns the row (line) position of the text cursor (0 means the top line).

## Requirements

CSRLIN is a library function to be included with the following command:

```
# include <csrlin.bas>
```

## Remarks

* This function is not available in Sinclair BASIC.
