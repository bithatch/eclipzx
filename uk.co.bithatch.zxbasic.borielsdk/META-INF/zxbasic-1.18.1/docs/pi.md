# PI

## Syntax

```
PI
```


## Description

PI is a math _constant_, roughly equivalent to **3.141592654...**,
and it is of type `Float`

## Example

```
REM Prints PI Value
PRINT "PI = "; PI
```
Which will print a _rounded value_ of `3.1415927`


## Remarks

*  This constant is 100% Sinclair BASIC Compatible.
